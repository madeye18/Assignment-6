Simple Web Server with Node.js A basic web server built with Node.js http module that demonstrates routing and serves HTML pages with CSS styling. How to Run

Ensure Node.js is installed on your system Navigate to the project folder in terminal Run command: node server.js Open browser and visit: http://localhost:3000

Routes

/ or /home - Home page (Status 200) /about - About page (Status 200) /service - Service page (Status 200) Any other route - Custom 404 error page (Status 404)

Features

Multiple route handling with proper HTTP status codes Modular code using helper functions Embedded CSS styling for all pages Custom 404 error page Request logging with timestamps Consistent navigation across pages Responsive design

Technologies Used

Node.js HTTP module HTML5 CSS3
